# compression index

**Category:** compressibility · **Influence (citation-weighted):** 87.09 across 12 papers

**Status:** 🆕 not yet derived · 🟢 non-leaky · 🔧 derivable from our inputs

*Also called:* Compression index; Compression index / compressibility; Compression index / compressibility (λ, κ); Compression index / lambda

## Derivation
Cc = 0.009*(LL-10) (Terzaghi & Peck) using USCS-keyed LL, or USCS-typical Cc (SP/SW ~0.02, ML ~0.1, CL ~0.25, CH ~0.5, OH/PT ~0.6-1.0). For overconsolidated/recompression use Cr~0.1-0.2*Cc.  [Terzaghi & Peck (1967); Skempton (1944)]

NON-leaky, but value comes only from USCS-keyed LL/Cc constants. Lower influence (87) and largely a deterministic function of USCS. Marginal but cheap; secondary. Not core, but recompression_index (its companion) is shortlisted as a representative compressibility feature.
