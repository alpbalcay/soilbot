# recompression index

**Category:** compressibility · **Influence (citation-weighted):** 27.25 across 4 papers

**Status:** 🆕 not yet derived · 🟢 non-leaky · 🔧 derivable from our inputs

*Also called:* Swell sensitivity (intrinsic/undisturbed swelling index ratio); Slope of swelling / recompression (unload-reload) line in e-ln p' space; Compression / swelling index (compressibility constants); compression / recompression index

## Derivation
Cr (swell/recompression index) ~ 0.1-0.2*Cc, with Cc from USCS or Cc=0.009*(LL-10). USCS-typical Cr: clean sand ~0, CL ~0.03-0.05, CH ~0.06-0.1, OH/PT higher.  [Terzaghi, Peck & Mesri (1996); Lambe & Whitman]
